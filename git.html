            GITHUB + VISUAL STUDIO CODE

            github.com : Create NEW REPOSITORY
            desktop.github : File -> Clone repository (выбрать локацию, куда клонировать, 
            например, на рабочий стол, там появится папка с проектом с репозитория)
            Открываем папку через VS Code
                
            VS Code : //кодим...
            desktop.github : отображаются все изменения -> Commit to ...branch -> 
                                Push (отправить коммит в репозиторий)
            github.com : обновить страницу, чтобы увидеть изменения

            Branch :: 
            desktop.github : отображаются все изменения -> Current branch -> New branch -> 
                                Create branch ->
                                Commit to new.branch -> 
                                Publish branch (отправит коммит в репозиторий новой веткой) ->
                                Create Pull Request - > 
                                перебрасывает на github.com : добавляем название и описание ->
                                Create Pull Request (создаем Pull Request) -> 
                                возвращаемся на страницу с кодом (на гитхабе):
                                    в ветке main ничего не поменялось, но появиласть new.branch с измененным кодом

            Merge ::
            github.com : вкладка Pull request -> выбираем имеющийся пул реквест -> 
                                Merge pull Request (слияние) -> 
                                Confirm merge (подтверждаем слияние) -> 
                                возвразаемся на страницу с кодом (на гитхабе):
                                    теперь наши изменения влились в ветку main

            Визуально увидеть все коммиты и ветки: вкладка Insights -> Network
